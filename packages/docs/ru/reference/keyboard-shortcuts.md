# Сочетания клавиш

Полная Figma-совместимая карта сочетаний клавиш. ✅ = реализовано.

::: tip Условные обозначения
⌘ = Cmd (Mac) / Ctrl (Win/Linux) · ⇧ = Shift · ⌥ = Alt/Option · ⌃ = Ctrl · ⌫ = Backspace
:::
